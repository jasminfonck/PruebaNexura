## Get started

Para clonar y ejecutar esto localmente, haga lo siguiente:

- git clone git@github.com :geshan/laravel-unit-test-example.git
- cp .env.ejemplo .env
- run composer install.
- php artisan serve
- Prueba http://localhost:8000
